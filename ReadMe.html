# Linktree Style Page ## Description Ce projet consiste à créer une page de type Linktree en utilisant HTML et CSS. La page est conçue pour servir de hub personnel, redirigeant les visiteurs vers différents contenus à travers une interface simple et élégante. Les dimensions de la page sont fixées à 366px de largeur et 800px de hauteur, optimisées pour une utilisation sur des appareils mobiles. ## Contenu du Projet ### Structure du Projet Le projet se compose de deux fichiers principaux : 1. `index.html` : Contient la structure HTML de la page. 2. `styles.css` : Contient les styles CSS appliqués à la page. ### Fonctionnalités - **En-tête** : Un texte de bienvenue centré en haut de la page. - **Photo de Profil** : Une image circulaire de 220px de diamètre avec une bordure blanche. - **Informations de Profil** : Affichage du nom d'utilisateur et du nom complet sous la photo de profil. - **Boutons de Navigation** : Trois boutons de navigation de couleur beige avec des bords très arrondis. - **Pied de Page** : Un texte en bas de page indiquant "@efp 2024". - **Fond Divisé** : Le fond de la page est divisé en deux couleurs, gris anthracite en haut et gris clair à partir de la moitié de la photo de profil. ### Aperçu ![Aperçu de la page Linktree](screenshot.png) ## Installation et Utilisation 1. **Cloner le dépôt** ```bash git clone https://github.com/yourusername/linktree-page.git
